Prework Study Guide

HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • An element refers to the entire element from opening tag to closing tag.
  • A tag refers only to what appears inside the angle brackets.

CSS

  • CSS adds style to the HTML framework.
  • There are three ways to style a webpage using CSS: inline CSS, internal CSS style sheet, and external CSS style sheet.
  • When a developer wants to style an element directly in the HTML file, they use inline CSS. Inline styles apply to a specific HTML tag, using a style attribute with a CSS rule to style a particular page element only.
  • A preceding period before a property designates it as a class selector. Any element assigned the class will gain the assigned properties.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git pull origin main: pulls down any changes from the base branch
